# RAG-with-RBAC
Private Chatbot for a Company

- Build internal chatbot for a company that can answer questions using company’s private data.
- Limit the access of information using Role Based Access Control(RBAC) which means only finance teams gets access to financial reports, marketing expenses. HR team can access employee data and payroll. Only C-level executives have full access to all company data.
- Protect sensitive data, out of scope detection.
